产品信息手册

智能客服系统 v2.0

核心功能
1. 智能问答：基于深度学习的自然语言理解，准确率95%以上
2. 多轮对话：支持10轮以上的连续对话，记忆上下文
3. 知识库对接：支持企业内部文档、FAQ等知识源
4. 人工转接：复杂问题可无缝转接人工客服

技术架构
1. 前端：Vue 3 + Element Plus，支持PC和移动端
2. 后端：FastAPI + Redis + PostgreSQL
3. AI模型：DeepSeek API + 自有微调模型
4. 部署：Docker + Kubernetes，支持弹性扩缩容

API接口
1. 对话接口：POST /api/chat
   - 参数：message, conversation_id, use_knowledge
   - 返回：response, context_used, timestamp
2. 历史接口：GET /api/conversations
   - 参数：page, page_size
   - 返回：对话列表，分页信息
3. 知识库接口：POST /api/knowledge/upload
   - 参数：file, type
   - 返回：处理结果，文档数量

使用说明
1. 登录系统：使用公司统一账号登录
2. 创建对话：点击"新建对话"按钮
3. 发送消息：在输入框输入问题，点击发送
4. 查看历史：在左侧历史记录中查看
5. 知识库管理：在"知识库"标签页上传文档

常见问题
Q：如何上传知识库文档？
A：在知识库管理页面，点击"上传文档"，选择txt或pdf文件即可。

Q：对话历史保存多久？
A：对话历史永久保存，但可以手动删除。

Q：是否支持多人协作？
A：暂不支持，正在开发中。
